数据库连接及增加记录

2020-03-05

1.在app.js中连接数据库
在app.js中增加初始化命令，并放置在onLaunch: function(){ ********* }中间。
wx.cloud.init({
env: '数据库的名字',
traceUser: true,
})

2.在页面的.js中连接数据库表
用这个语句
const DB = wx.cloud.database().collection("*******")
在所需要的页面中连接数据库表
放置在Page({ })的外面

3.编写函数
放置在Page({ })的内部
Page({
addData(){
DB.add({
data:{
phone: "12345678901",
name: "worker1",
carNumber: "BA0143"
},
success(res){
console.log("添加成功",res)
},
fail(res) {
console.log("添加失败", res)
}
})
}
})
